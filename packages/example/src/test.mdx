# A live code block

```jsx live
import { useState } from "react";

() => {
    const [visible, setVisible] = useState(false); 
    return (
        <div>
            <div>{visible.toString()}</div>
            <button onClick={() => setVisible(visible => !visible)}>Toggle</button>
        </div>
    )
}
```

## Isolated Scopes

The following two code blocks both import named exports from different modules, but the named exports have the same 
name. This demonstrates that the scopes for the two live code blocks are isolated from each other.

```jsx live
import { Button } from "module-a"

() => {
    return <Button>Click Me</Button>
}
```

```jsx live
import { Button } from "module-b/Button.jsx"

() => {
    return <Button>Click Me</Button>
}
```
